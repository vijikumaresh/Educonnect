## How It Works Automatically

### When You Click on Any Folder:

1. **Sidebar**: You click on any folder (at any nesting level)
2. **Main Area**: The folder name appears as the title
3. **Add Contact Button**: Automatically appears in top-right
4. **Click Add Contact**: Form opens
5. **College Name**: Auto-fills with the folder name
6. **Save**: Student is saved to that specific folder
7. **View**: Student appears in the table for that folder only

### Visual Flow:

```
Click Folder          Main Area Updates              Form Auto-fills
─────────────        ───────────────────            ────────────────
                     
📁 Dindigul    →     ┌─────────────────────┐       College Name:
    (click)          │ Dindigul  [+Add]    │   →   "Dindigul"
                     └─────────────────────┘       (auto-filled)

📁 MIT College →     ┌─────────────────────┐       College Name:
    (click)          │ MIT College [+Add]  │   →   "MIT College"
                     └─────────────────────┘       (auto-filled)

📁 Comp Sci    →     ┌─────────────────────┐       College Name:
    (click)          │ Comp Sci [+Add]     │   →   "Computer Science"
                     └─────────────────────┘       (auto-filled)
```
