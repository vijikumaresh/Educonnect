Common Issues & Solutions

  1. Backend not accessible: Make sure backend is running on cargo run
  2. CORS errors: Backend should allow requests from your frontend URL
  3. Wrong API URL: Check frontend/src/services/api.ts
  4. Port mismatch: Backend should be on 8080, Frontend on 5173 or 3000
